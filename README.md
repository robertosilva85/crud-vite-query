# CRUD Vite React Query

I am creating a small project to start with Vite and React Query and creating a generic hook to get any query from API.

## What is using here?

- Material-UI
- React Query
- Vite Tools
- Axios

## Running

```
  git clone git@github.com:robertosilva85/crud-vite-query.git
  cd crud-vite-query
  npm install
  npm run dev
```
